Level 1
1. Create a springboot application with GET, POST or DELETE method type endpoints. Deploy and verify the outcome via Postman.
2. Create 2nd project with some  endpoints and access the first project using Rest Template. (Interaction between Microservices)
3. Create a project as Service Discovery i.e Eureka Server and register the above created Microservices on Eureka. Access and verify the outcome.

Level 2
1. Create a separate project acting as a Config Server(Config Microservice) and access these configurations by other Microservices.
2. Enable spring security in one of the project of Level 1 and verify the basic authentication provided by SpringBoot.
3. Create a full fledged project comprising of Controller, interface, implementation, Repository 
   and Entity class and use JPA repository for interaction with DB.
   
Level 3 
1. JWT and OAuth implementation
2. Centralized Logging
3. Apache Kafka with Microservices.
4. Dockerizing a Microservice and deploying on Azure/ AWS Cloud.
